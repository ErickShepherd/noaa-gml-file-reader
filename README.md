# noaa-gml-file-reader

Read NOAA GML atmospheric trace-gas ASCII data files as `pandas.DataFrame` objects.

> **Lineage:** formerly `noaa_esrl_gmd_file_reader`. NOAA renamed ESRL's Global Monitoring
> *Division* (GMD) to the Global Monitoring *Laboratory* (GML) in 2020; current data lives at
> [gml.noaa.gov](https://gml.noaa.gov). The package (import name `noaa_gml_file_reader`,
> distribution `noaa-gml-file-reader`) was renamed to match. This is a v2 stub — the full
> README (usage, supported dialects, breaking-change notes) lands later in the v2 work.
